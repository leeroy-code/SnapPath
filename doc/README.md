# SnapPath

> 为 AI 编程而生的 macOS 截图工具。截图、保存、路径自动复制到剪贴板，一气呵成。

## 它解决什么问题？

AI CLI 工具（Claude Code、Cursor、Aider）运行在终端中，无法粘贴图片。当你需要把报错截图、UI 问题发给 AI 时，你需要的是**文件路径**，而非图片本身。

SnapPath 的工作流程：

```
按下快捷键 → 截图自动保存到本地 → 文件路径自动复制到剪贴板 → 终端里 Cmd+V 粘贴路径
```

## 核心功能

- **四种截图模式**：区域选取、全屏、窗口、Pin 区域
- **全局快捷键**：任何应用中均可触发（默认 `⌘⇧S` / `⌘⇧A` / `⌘⇧W`）
- **路径自动复制**：截图后文件绝对路径立即进入剪贴板
- **菜单栏常驻**：无 Dock 图标，不干扰工作流
- **多显示器支持**：区域选择和窗口选择支持跨多显示器操作

## 截图编辑器

截图后自动打开编辑器（可在设置中关闭），支持以下标注工具：

- **箭头**：绘制箭头指示重点区域
- **矩形框**：用矩形框圈出关键内容
- **文字**：添加文字说明，点击已有文字可重新编辑
- **裁剪**：裁切掉不需要的部分

编辑器还支持：
- 自定义标注颜色和字号
- 撤销操作（⌘Z）
- 缩放查看（滚轮或工具栏滑块）
- 适应窗口（⌘0）

完成编辑后，可选择：
- **Copy Path**：保存截图并复制文件路径到剪贴板
- **Copy Image**：直接复制图片到剪贴板（不保存文件）
- **Pin**：将截图固定到屏幕上

## Pin 功能

使用 `⌘⇧P` 可快速截取区域并固定到屏幕上，适合临时参考内容。

Pin 窗口特性：
- 始终置顶显示，可跨所有桌面
- 滚轮缩放（0.25x - 4x）
- 拖拽移动位置
- 右键菜单：复制图片、保存并复制路径、调节透明度（25%/50%/75%/100%）
- 双击或按 ESC 关闭

## 系统要求

- macOS 11.5+
- 需要授予屏幕录制权限
