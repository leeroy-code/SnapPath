# 使用指南

> 截图模式、快捷键和设置项的完整说明。

## 截图模式

### 区域截图

通过 `⌘⇧S` 或菜单 "Region Capture" 触发。

- 屏幕覆盖半透明遮罩，光标变为十字准星
- 按住鼠标拖拽选择区域，白色边框标示选区
- 多显示器环境下遮罩会覆盖所有屏幕；选区默认锁定在开始拖拽的那块屏幕（拖到屏幕外会自动贴边）
- 松开鼠标完成截图
- `ESC` 取消

### 全屏截图

通过 `⌘⇧A` 或菜单 "Full Screen Capture" 触发。

- 截取鼠标当前所在的屏幕
- 多显示器环境下，只截取鼠标所在的那一个屏幕
- 触发后立即完成，无需额外操作

### 窗口截图

通过 `⌘⇧W` 或菜单 "Window Capture" 触发。

- 进入窗口选择模式，屏幕显示半透明遮罩
- 鼠标悬停时，对应窗口显示高亮
- 点击选中窗口完成截图
- `ESC` 取消

### Pin Region（钉住区域）

通过 `⌘⇧P` 触发。

- 选择区域后，截图会作为浮动窗口固定显示在屏幕最前端
- 浮动窗口支持以下操作：
  - **拖动**：点击窗口任意位置拖动
  - **缩放**：滚轮上下滚动调整大小（25% - 400%）
  - **透明度**：右键菜单可设置透明度（100%/75%/50%/25%）
  - **复制图片**：右键菜单 "Copy Image" 复制图片到剪贴板
  - **保存并复制路径**：右键菜单 "Save & Copy Path" 保存文件并复制路径
  - **关闭**：按 `ESC`、双击窗口或右键菜单 "Close"
- 适合需要参考某个区域内容时使用
- 点击窗口右上角的绿色 Pin 图标也可快速关闭窗口

## 截图编辑器

当"截图后显示编辑器"设置开启时（默认开启），截图完成后会弹出编辑窗口。

### 标注工具

编辑器提供以下标注工具：

| 工具 | 说明 |
|------|------|
| 箭头 (Arrow) | 绘制带箭头的指示线 |
| 矩形 (Rectangle) | 绘制矩形边框，用于高亮区域 |
| 文字 (Text) | 添加文字标注，点击已有文字可重新编辑 |
| 裁剪 (Crop) | 裁剪图片，仅保留选中区域 |

### OCR 文字识别

编辑器提供 OCR 功能，可识别截图中的文字内容：

- 点击工具栏的 OCR 按钮（文字识别图标）
- 自动识别图片中的文字（支持中英文）
- 识别结果自动复制到剪贴板
- 识别成功后自动关闭编辑器窗口

### 编辑器功能

- **颜色选择**：点击颜色选择器更改标注颜色
- **字体大小**：下拉菜单选择文字标注的字号（12pt - 48pt）
- **撤销**：撤销上一步操作（支持多步撤销），快捷键 `⌘Z`
- **缩放**：使用触控板双指缩放或鼠标滚轮调整画布缩放（10% - 400%）
- **取消**：放弃编辑，按 `ESC` 或 `⌘W` 或点击 "Cancel"
- **复制图片**：仅复制图片到剪贴板（不保存文件），快捷键 `⌘C` 或 `Enter`
- **复制路径**：保存文件并复制路径到剪贴板，快捷键 `⌘S`
- **钉住**：将编辑后的图片作为浮动窗口钉住
- **OCR**：识别图片中的文字并复制到剪贴板

## 截图后的行为

每次截图完成后，SnapPath 自动执行：

1. **保存文件** — PNG 格式，文件名 `screenshot_YYYY-MM-DD_HH-mm-ss.png`
2. **复制路径** — 文件的绝对路径复制到系统剪贴板
3. **发送通知** — 系统通知显示"截图已保存"及文件路径（可关闭）
4. **播放音效** — 系统提示音（可关闭）

## 快捷键

| 操作 | 默认快捷键 |
|------|-----------|
| 区域截图 | `⌘⇧S` (Command + Shift + S) |
| 全屏截图 | `⌘⇧A` (Command + Shift + A) |
| 窗口截图 | `⌘⇧W` (Command + Shift + W) |
| Pin 截图 | `⌘⇧P` (Command + Shift + P) |
| 复制 Finder 路径 | `⌘⇧C` (Command + Shift + C) |

快捷键为全局快捷键，在任何应用中均可触发。

**复制 Finder 路径**：在 Finder 中选中文件后按下 `⌘⇧C`，即可将选中文件的绝对路径复制到剪贴板。支持多选文件，多个路径会以换行分隔。

**自定义快捷键**：打开设置窗口（菜单栏图标 → Settings...），在 Keyboard Shortcuts 区域点击录制按钮，按下新的快捷键组合即可。

支持自定义的快捷键：
- Region Capture（区域截图）
- Full Screen Capture（全屏截图）
- Window Capture（窗口截图）
- Pin Region（钉住区域）
- Copy Finder Path（复制 Finder 路径）

## 设置

通过菜单栏图标 → "Settings..." 或 `⌘,` 打开设置窗口。

| 设置项 | 说明 | 默认值 |
|--------|------|--------|
| Save Location | 截图保存目录，点击 "Choose..." 更改 | `~/Downloads` |
| Launch at Login | 开机自动启动 | 关闭 |
| Play sound effect | 截图成功后播放提示音 | 开启 |
| Show notification | 截图成功后显示系统通知 | 开启 |
| Show editor after capture | 截图后自动打开编辑器 | 开启 |
| Auto check for updates | 自动检查应用更新 | 开启 |
| Language | 界面语言（English / 中文）| English |

## 菜单栏菜单

点击菜单栏相机图标，显示：

```
Region Capture
Full Screen Capture
Window Capture
─────────────────
Pin Region
─────────────────
Copy Finder Path
─────────────────
Open Screenshots Folder
─────────────────
Settings...          ⌘,
Check for Updates
─────────────────
About SnapPath
Quit SnapPath        ⌘Q
```

**Copy Finder Path**：复制 Finder 中当前选中的文件路径到剪贴板。

**Open Screenshots Folder**：在 Finder 中打开当前截图保存目录。

**Check for Updates**：手动检查应用更新。

**About SnapPath**：显示应用版本和版权信息。
